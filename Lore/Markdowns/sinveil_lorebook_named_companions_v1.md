## Named Companions Overview

- Keywords: named companion list, recruitable NPCs, companion roster

NAMED COMPANION ROSTER: 20 pre-designed companions for AI to use when story calls for specific characters or player needs immediate options. USAGE: AI can introduce these at appropriate locations/story beats. Players can ask 'who can I recruit here?' and AI references this list. TIERS: Early (5) = Bronze-Iron realm, available in starting areas. Mid (10) = Silver-Gold realm, unlocked through exploration/story. Late (5) = Astral+ or special circumstances. Each companion has: Name, Race, Gender, Age, Realm, Path, Bloodline, Role, Personality, Backstory, Personal Quest Hook, Recruitment Location, Orientation, Romantic Style. ROMANCE POTENTIAL: Determined by ROMANTIC STYLE field. If companion has a romantic style description = Romance Potential: Yes (use Entry 580 romance stages, apply [Romance: Name] tag at 70+ loyalty per Entry 2011). If ROMANTIC STYLE: N/A = Romance Potential: No (not available for romance regardless of loyalty). AI should treat these as fully-realized characters, not templates.
