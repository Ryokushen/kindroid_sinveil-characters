# PDF Text Extractions

This folder contains OCR-extracted text from the Kindroid documentation PDFs.

## Files:

- **7DeadlySins_OCR.txt** - Common Kindroid setup mistakes and how to avoid them
- **BreakYourAI_OCR.txt** - Cautionary guide on AI reinforcement and memory management
- **BuildingACustomKindroid_OCR.txt** - Beginner's guide to creating AI companions
- **ElevenlabsV3_OCR.txt** - Voice creation guide for ElevenLabs V3 integration
- **KindroidBehavior_OCR.txt** - Tips for maintaining consistent Kindroid behavior
- **KindroidDynamism_OCR.txt** - Understanding and adjusting dynamism settings
- **KindroidEffectiveBS_OCR.txt** - Tools and tips for crafting effective backstories
- **KindroidFix_OCR.txt** - Troubleshooting guide for fixing broken Kindroids
- **KindroidFoundation_OCR.txt** - Building solid foundations for lasting Kindroids
- **KindroidJournals_OCR.txt** - Understanding and using journal entries effectively
- **KindroidJournals2_OCR.txt** - Advanced journal techniques for lore and worldbuilding
- **KindroidPersonality_OCR.txt** - Creating memorable AI personalities with depth
- **KindroidPersonas_OCR.txt** - User personas and identity management
- **KindroidTiers_OCR.txt** - Guide to Kindroid subscription tiers and memory expansion
- **KindroidV7GC_OCR.txt** - Group chat functionality in Kindroid V7
- **KindroidWorldbuilding_OCR.txt** - Foundations of worldbuilding using Kindroid
- **V6PartII_OCR.txt** - Stable bodies, hair, and physical features for V6 selfies
- **V6Selfies_OCR.txt** - Visual character creation guidance

## Extraction Method:
All text was extracted using OCR (Optical Character Recognition) from the original PDF files using a custom Python script with pdfplumber and pytesseract.

## Usage:
These extracted texts were used to understand proper Kindroid character creation principles for designing female characters in the Sinveil cultivation world setting.
